TravelGo - Admin Panel
The TravelGo Admin Panel, powered by ReactJS and Material-UI (MUI), streamlines train schedule management. Leveraging Nivo for dynamic data visualizations, Formik for seamless form handling, and Redux Toolkit for state management, it offers a robust and user-friendly interface. With comprehensive testing using Jest and Supertest, this panel ensures code integrity. It empowers administrators with insightful data, facilitating informed decisions and enhancing operational efficiency.
Source code: https://github.com/movinsilva/TravelGo-AdminPanel
